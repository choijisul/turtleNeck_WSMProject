## __🐢 | 어떤 웹사이트인가?__<br>
<pre>

## __✏️ | 기획 배경__<br>
<pre>
아무래도 미림의 학생들은 오랫동안 컴퓨터를 해야하는 학교의 특성상 팔목, 허리 등
다양한 신체에 불편함이 있을 것이라고 생각하여 이러한 웹사이트를 기획하게 되었습니다.
</pre>
